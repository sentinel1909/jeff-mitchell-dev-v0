Projects


There has been many things I've fiddled with in my journey to learn Rust. Here are some of the highlights:


This Site

I've been up and down and back and forth with a “developer profile” site over the years. What you see here is latest iteration. It's built with Axum and Askama and hosted on Shuttle. The GitHub repo is here: jeff-mitchell-dev


Zero to Production in Rust

In 2022, I worked through Zero to Production in Rust, by Luca Palmieri

Rather than just work through the book as it was, I swapped the web framework for Axum instead of Actix Web. For a time, I had the code running in production on Railway, but decided to take it down for a time and eventually work on missing features.

The GitHub repo is here: Crusty Rustacean API


NASA Astronomy Photo of the Day

I made a site with the Yew Web framework. It retrieves and displays a daily photo (or video) from the NASA Astronomy Photo of the Day API

NASA APOD Viewer


Rivet Head API

In the fall of 2023 I started an API project to power a music listening diary. It's running now in basic form. I'm making this for myself, it won't be available to the public. So far, I interact with it via cURL. I'm intending on integrating it with this site, to power the Music page.

I recently swapped the Actix Web framework for Axum. It uses Askama for templating, enabling responses in HTML, in order to more easily use HTMX on the front end.

The Github repo is here: Rivet Head API


Rivet Head CLI

Interacting with my Rivet Head API is tedious when using cURL. I set out to make a simple CLI, which cuts the typing down by quite a lot. The CLI app used the clap crate as well as the reqwest crate

The GitHub repo is here: Rivet Head CLI


Shuttle Christmas Code Hunt

Since my coding adventure began in early 2021, I've attempted to complete the annual Advent of Code challenge that runs each year in December. I've never gotten very far. In December 2023, the folks at Shuttle decided to host their own incarnation, called the “Christmas Code Hunt”. I initially tried completing the challenges without a full web framework, using Tower and building from scratch. The water was pretty deep and I didn't get particularly far. I restarted using the Axum web framework, and maintain a repo where I can return and continue to work on the challenges.

The Github repo is here: Shuttle Christmas Code Hunt


Fun with Nom

While doing the Shuttle Christmas Code Hunt in December 2023, I dove headlong into the Nom crate. I returned for a deep dive at the beginning of 2024, starting a project I christened “Fun with Nom”. I created an API with one endpoint that you can send a phrase to and get the word count back.

The Github repo is here: Fun with Nom
